# Sistema de Eleição em Java 🗳️

Este é um projeto acadêmico desenvolvido em **Java** para colocar em prática os conceitos de **Programação Orientada a Objetos (POO)**. O sistema simula o funcionamento básico de uma eleição.

## Funcionalidades
* **Cadastro de Candidatos:** Permite registrar quem vai receber os votos.
* **Cadastro de Eleitores:** Registra as pessoas que estão aptas a votar.
* **Votação:** Computa os votos de cada eleitor de forma organizada.
* **Apuração:** Mostra o resultado final da eleição com a contagem dos votos.
